# Hotel Booking Analytics System

## Problem Statement

Hotels generate large amounts of booking data. This project analyzes booking patterns, pricing trends, guest behavior, and cancellation factors to help improve business decisions.

## Objectives

- Analyze booking demand
- Study pricing trends (ADR)
- Identify peak seasons
- Analyze guest preferences
- Understand cancellation behavior

## Tech Stack

- Python
- Pandas
- Matplotlib
- Streamlit (planned)
- Scikit-Learn (planned)
